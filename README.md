# ThreeFivePlayground

**Xamarin.Forms** sample to show the main features of **3.5** version.

The sample is available for two platforms:

- iOS
- Android

Features:
- Bindable Layout
- ListView ScrollBarVisibility
- SwitchCell OnColor
